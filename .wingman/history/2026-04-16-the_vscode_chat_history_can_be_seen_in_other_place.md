# The VSCode chat history can be seen in other place

**User**
The VSCode chat history can be seen in other places? For example: in GitHub?

**Copilot**
No, No, the VS Code chat history from the Copilot Chat panel is not automatically visible in other places like GitHub. The chat history is stored locally in your VS Code environment and is not published or synced to GitHub or any other external service by default. If you want to share your chat history, you would need to manually copy and paste it or export it yourself.

**User**
How could I make that copy-paste?

**Copilot**
To copy and paste your VS Code chat history:

1. Open the Copilot Chat panel in VS Code.
2. Select the text you want to copy by clicking and dragging your mouse over the chat messages.
3. Right-click the selected text and choose "Copy" (or press Ctrl+C).
4. Go to your desired destination (e.g., a GitHub issue, pull request, or document).
5. Paste the copied text (Ctrl+V).

This will transfer your chat history wherever you need it.

**User**
Ok, this is a manual copy, but is there some built-in bulk chat export option?

**Copilot**
No, there is currently no built-in bulk export option for Copilot Chat history in VS Code. You need to manually copy the chat if you want to export or share it.

**User**
Is there a plug-in or extension for this? |Or else, for synchronizing?

**Copilot**
There is currently no official Copilot Chat extension or plugin that provides automatic chat export or synchronization with external services like GitHub. The Copilot Chat history is designed to remain local to your VS Code environment.

However, you can search the VS Code Marketplace for third-party extensions that might offer chat export, backup, or synchronization features. As of now, such functionality is not standard or widely available for Copilot Chat specifically.
